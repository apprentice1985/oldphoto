Introduction

A web site for share photo, powerd by Django.

You can get the source from svn(http://oldphoto.googlecode.com/svn/trunk/), and see the demo on http://www.lzpian.com/.
Requirements

   1. Python
   2. Django
   3. PIL 

Install

    * Change the db setting in settings.py, and run syncdb.bat to init db.
    * Change MEDIA_ROOT to set the resource path, where the uploaded photo will save. 

介绍

代码在SVN中(http://oldphoto.googlecode.com/svn/trunk/)。

所需组件

   1. Python

   3. PIL 

安装

    * 修改MEDIA_ROOT将其配置成资源文件路径，上传的图片文件将保持在该路径中。
